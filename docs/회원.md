# 회원가입

```
1. 이메일,비밀번호, 이름,전화번호,주소,성별을 입력받는다.
2. 전화번호는 , 중복 불가능 / 암호화
    - 회원가입 폼에서 인증
    - 중복 불가
    - 암호화 저장
3. 주소
    - 도로명주소
    - 상세주소
4. 이메일
    - 회원가입 완료 후 확인 메일을 통해 최종 생성
    - 암호화 저장
    - 중복 불가
5. 비밀번호
    - Bcrypt 해시 처리

6. Google / Kakao 로그인 (후순위)
```

# 로그인

```
1. JWT Access Token
    - 유효기간 10분
    - Bearer Header로 전달
2. Refresh Token
    - 유효기간 30분
    - Http Only Cookie 전달
    - in-memory db에 저장
3. 로그인 5회 실패 시 계정 잠금
```
